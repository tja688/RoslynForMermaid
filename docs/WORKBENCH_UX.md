# ArchRadar Workbench UX Notes

## Navigation
- Back / Forward: step through view history.
- Up: clear selection or move up one layer.
- Home (L0): return to the top layer.
- Breadcrumb chips show Project / Snapshot / Layer and allow quick layer jumps.

## Canvas Controls
- Scroll to zoom, drag background to pan.
- Fit and Reset buttons in the canvas toolbar.
- Double-click background to reset view.

## Interaction Feedback
- Hover highlights clickable nodes/edges and shows labels.
- Selected nodes/edges are highlighted and mirrored in the Inspector.
- Click blank space to clear selection.

## Layout & Panels
- Three-column workbench: left (data + config), center (canvas), right (inspector).
- Sidebars are collapsible and independently scrollable.
- Status messages appear as floating toasts.

## Config Panel
- Grouped sections with helper text and validation.
- Read-only fields are non-interactive.
- Save updates config; scan is explicitly triggered via Start Scan.
